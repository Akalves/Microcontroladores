
Q1. Quais as diferenças entre os barramentos de dados e de endereços?
O primeiro tem por finalidade a troca de dados(informações) da CPU com unidade de execução do comando realizado, realizando um caminho
fixo para essa troca, no segundo, o barramento de endereços tem por objetivo a indicação do endereços de armazenamento dos dados a ele
imputados para futura solicitação da CPU.

Q2. Quais são as diferenças entre as memórias RAM e ROM?
Memória RAM é volátil, sua capacidade de armazenar de dados é temporária, o que no caso da memória ROM difere, esta tem sua construção 
no objetivo de armazenar dados permanentes e modificaveis mesmo após seu desligamento

Q3. Considere o código abaixo, de 12 linhas:

    1.   #include <stdio.h>

    3.   {
    4.      int i;

    6.      scanf("%d", &i);
    7.      if(i%2)
    8.         printf("%d eh impar.\n");
    9.      else
   10.         printf("%d eh par.\n");
   11.      return 0;
   12.   }

Para este código, responda:
	(a) A variável 'i' é armazenada na memória RAM ou ROM? Por quê?
	Na memória RAM, pois esta ainda é uma alocação do projeto programacional
	(b) O programa compilado a partir deste código é armazenado na memória RAM ou ROM? Por quê?
	Na memória ROM, pois quando compilado o programa, este se encontra na ROM como base do programa
Q4. Quais são as diferenças, vantagens e desvantagens das arquiteturas Harvard e Von Neumann?
Basicamente a de Von Neuman tem seu processamento e amrmazenamento no memsmo conjunto, enquanto a  de Havard se distingue por ter seu
processamento e armazenamento separados, possibilitando a execuaçaõ de tarefas distintas entre dados e memórias.

(a) Little-endian?
